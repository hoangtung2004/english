# English Learning Web App

## Cài đặt

1. Clone project:
   ```
   git clone <link-repo>
   cd <tên-thư-mục-project>
   ```

2. Cài đặt dependencies:
   ```
   npm install
   ```

3. Tạo file `.env` từ mẫu:
   ```
   cp .env.example .env
   ```
   (Chỉnh sửa thông tin kết nối MongoDB nếu cần)

4. Chạy project:
   ```
   npm start
   ```
   hoặc
   ```
   node src/app.js
   ```

5. Truy cập: [http://localhost:3000](http://localhost:3000)
